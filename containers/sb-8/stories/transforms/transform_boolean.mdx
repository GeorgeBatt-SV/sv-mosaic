# transform_boolean

Converts boolean into more friendly formats.

#### Conversions

* `true` => `"Yes"`
* `false` => `"No"`

#### Args

None

#### Example

```js
import { transform_boolean } from "@simpleview/sv-mosaic"
...
transforms : [
    transform_boolean()
]
```
